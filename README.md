# Take A Wok — Landing Page

Landing page oficial para **Take A Wok**, el mejor concepto de comida wok en la Ciudad de México.

## Descripción

Sitio web de una sola página (single-page) que presenta el menú, sucursales, opciones de entrega a domicilio e información sobre franquicias de Take A Wok.

## Tecnologías

- HTML5
- CSS3
- JavaScript vanilla
